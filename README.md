# Lottie Like Animation
this is a like button with a colorfull animation, very like the twitter like button on the feed.

![app screenshot](./app_screenshot.png)

## Built with
- expo
- react native
- lottie react native
- json

### Demo
[video demo](https://www.youtube.com/watch?v=jO7cL1v4QIQ)

### Available Scripts

First of all clone this repository.

In the project directory, you can run:

- step 1: run `npm install` (wait untill all node-modules are in place) 

- step 2: run `npm start` (open the project in an emulator or a real mobile)
